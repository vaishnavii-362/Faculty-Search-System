#Faculty Search System

A simple Python project that helps VIT students search for faculty details during FFCS.
The system allows keyword-based search and lets users add new faculty entries.
It uses only basic Python concepts and stores data in a separate module (faculty_info.py).

*Features*
Search faculty by:
  1. Name
  2. Course code
  3. Slot
Supports partial/keyword search
Add new faculty entries
Separate data file for easy editing
Beginner-friendly (no JSON, no databases)

*File Structure*

faculty-search-system/
│
├── main.py              # Main program with menu & logic
├── faculty_info.py      # Contains faculty data
└── README.md            # Project documentation

*How to Run*
1. Download or clone the repository
2. Open the folder
3. Run the program:

    python main.py

*Example Usage*

Menu:
1. Search Faculty
2. Add Faculty
3. Exit

Search Example:
Input keyword: Sasmita
Output: Dr. Sasmita Padhy – CSE2001 – C14+E11+E12 – sasmita.padhy@vitbhopal.ac.in – Rating: 4.2
